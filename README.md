# LocalFix – Home Service Booking System

## Abstract

LocalFix is a web-based home service booking platform developed using the MERN stack (MongoDB, Express.js, React.js, and Node.js) with Tailwind CSS for frontend design and Clerk for authentication. The system is designed to streamline the process of booking and managing home services by connecting customers with service providers in a structured and efficient manner.


## Features

### Customer

* User registration and authentication using Clerk
* Browse available services
* Book a service with address and details
* Track booking status (pending, accepted, in-progress, completed)
* Make payments (Payment Integration not implemented)
* Submit ratings and feedback

### Service Provider (Technician)

* Login and role-based access
* Manage availability
* View and accept service requests
* Update job status
* Track completed services and performance

### System Features

* Role-based access control (Customer / Provider)
* Secure authentication using Clerk
* RESTful API architecture
* Booking lifecycle management
* MongoDB database integration
* Responsive UI using Tailwind CSS


## Tech Stack

### Backend
